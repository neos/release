Introduction Package

The TYPO3 Introduction Package guides users the the installation and setup of a default website with a clean XHTML template and dozens of pages that can be customized with colors and logos. It can be used to build a first website or just to get a broad overview of the main features and best practices in TYPO3. Just download, install, and see how easy it is to start building a TYPO3 website.

Download TYPO3 version 4.5 LTS with the new Introduction Package

You need a server that meets the minimum system requirements to fully try out the TYPO3 Introduction Package. If you want to test the TYPO3 Introduction Package on your local computer instead, please download one of the Installers first.

Packages

Which package to use

  • Introduction Package: This is an example of a typical TYPO3 installation that bundles an XHTML template, the TYPO3 source and some well-known extensions. Download this if you want to try out TYPO3 and play around with it, or just to get a basic site with a predefined template up and running. To install, you need a webserver on your local machine or a hosted webserver.
  • Source + Dummy: This is a skeleton site and the TYPO3 source. Download this if you're going to build a new site from scratch.
  • Source: This is just the source of TYPO3. Download it if you have a running TYPO3 installation and would like to upgrade to a new version.
  • Dummy: This is just a skeleton site. Download it if you already have the TYPO3 source and just want to set up a new site.

TYPO3 Versions at a Glance

TYPO3 4.5 The latest stable release (for new and existing projects)
This version is the current stable release. It is recommended for existing projects and will get full support until April 2014.
TYPO3 4.4 The old stable release (for old projects)
This version is the old stable release. It can be used for existing projects, but gets only security fixes. Updating to TYPO3 4.5 is recommended.
TYPO3 4.3 Outdated stable release
Support for this version will be dropped entirely when TYPO3 4.6 is released. Updating to TYPO3 4.5 is seriously recommended.
TYPO3 4.2 Obsolete stable release
Support for this version is completely dropped since the release of TYPO3 4.5. Updating to TYPO3 4.5 is strongly recommended.

For more details on support for stable and older versions of TYPO3, please refer to the TYPO3 v4.x maintenance policy.

Older Releases

Older releases can be found on Sourceforge.net.

Installers

These installers install a complete Apache / MySQL / PHP environment with a TYPO3 package for testing locally on your computer.

The installers don't necessarily contain the latest versions programs, so you will need to update them yourself to make your installation secure. The purpose of these installers is to get you started as quick as possible. They are intended for testing, but must not be used for any production use!

Please note that we don't offer support for any of these packages!
